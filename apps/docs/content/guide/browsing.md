# 浏览版面和主题

## 查找版面

点击顶部的“版面列表”，可以按分区浏览全部版面。进入版面后，主题列表会显示标题、作者、回复情况和最后更新时间。分页、筛选和列表状态会保留在地址中，刷新页面或分享链接后仍能回到同一位置。

知道版面名称时，也可以把顶部搜索类型切换为“版面”，直接搜索名称。

## 阅读主题

点击主题标题进入阅读页。一个主题有多页时，可以使用页面上方或下方的分页。楼层链接带有独立地址，打开后会定位到对应楼层。

阅读页可能包含 Markdown 或旧版 UBB 内容。图片、链接、代码块、引用、媒体和公式会使用统一样式展示；浏览器不允许访问的地址会降级为普通文本。

## 使用发现入口

| 入口 | 适合查看的内容                     |
| ---- | ---------------------------------- |
| 新帖 | 全站最近更新的主题                 |
| 精选 | 经过推荐的主题                     |
| 关注 | 关注版面、关注用户和收藏主题的更新 |
| 热门 | 本周、本月或历史热门主题           |

“新帖”支持经典、卡片和媒体三种显示方式。选择会保存在浏览器中，下次打开时继续使用。

## 搜索主题和用户

顶部搜索框支持“主题”“用户”“版面”三种全站搜索。在版面页或主题页内，还会出现“版内”选项。输入关键词后按回车或点击搜索按钮即可。

搜索结果为空时，可以缩短关键词，或确认当前是否限制在某个版面。查看帖子后想发表内容，可以继续阅读[发主题、回复和编辑](/guide/writing)。
