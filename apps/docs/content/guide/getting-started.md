# 登录并开始使用

## 登录 CC98

打开论坛后，点击顶部右侧的“登录”。输入 CC98 用户名和密码，登录成功后会回到刚才访问的页面。需要注册账号时，点击“注册”前往统一账号页面。

登录状态过期后，论坛会要求重新登录。重新登录前，当前页面地址会被保留，完成登录后可以继续之前的操作。

> 密码只应填写在 CC98 的登录页面。不要把密码、登录令牌或浏览器 Cookie 发给其他人。

## 安装到桌面

支持 PWA 的浏览器可以把 CC98 安装到桌面或主屏幕。安装入口通常位于地址栏或浏览器菜单中，名称可能是“安装应用”“添加到主屏幕”或“创建快捷方式”。安装后会以独立窗口打开，账号和外观设置仍由当前浏览器管理。

页面提示新版本已经准备好时，点击“刷新”即可切换版本。选择“稍后”不会影响当前页面，下次发现更新时还会再次提示。

## 认识顶部导航

登录后，顶部区域提供这些常用入口：

- **版面列表**：按分区查看全部版面。
- **新帖**：查看全站最近发布或回复的主题，并切换列表样式。
- **关注**：查看关注版面、关注用户和收藏主题的更新。
- **精选**：查看推荐内容。
- **搜索**：搜索主题、用户和版面。在版面或主题页内搜索时，还可以选择“版内”。
- **消息**：进入回复、提及、系统通知和私信。
- **个人中心**：管理自己的主题、回复、收藏、关注、历史、版面和外观设置。

## 退出登录

点击顶部右侧的用户名，在菜单中选择“退出登录”。退出后，公开内容仍可浏览，需要账号的页面会再次要求登录。
