# 发主题、回复和编辑

## 发表新主题

进入目标版面，点击“发表主题”。登录后才能进入编辑页面。

填写标题并选择内容类型。正文使用 Markdown 编辑器，可以直接输入文字，也可以使用工具栏插入标题、粗体、斜体、链接、列表、引用和代码。提交前确认标题、版面和正文内容无误。

## 回复主题

在主题阅读页找到回复区域，输入内容后提交。需要引用某一楼时，使用该楼层提供的回复入口，编辑器会带入对应的引用信息。

提交期间不要连续点击按钮。成功后页面会显示新楼层；失败时正文会保留，可以按页面提示重试。

## 上传图片和附件

编辑器支持上传图片和附件。选择文件后，等待上传完成并确认正文中出现对应链接，再提交帖子。

上传失败时可以检查：

- 网络是否稳定；
- 文件大小和类型是否符合页面提示；
- 登录状态是否仍然有效；
- 当前版面是否允许发表内容。

不要在上传尚未完成时关闭页面。写入失败后如果需要刷新，可以先复制正文，避免浏览器异常导致草稿丢失。

## 编辑已经发表的内容

帖子作者或有对应权限的管理人员可以看到“编辑”入口。编辑页面会载入原有内容，保存成功后返回主题页。

旧版 UBB 帖子进入新编辑器时会转换为 Markdown。转换后先检查引用、链接、图片和复杂排版，再保存修改。

## 常用 Markdown

```markdown
# 标题

**粗体**、*斜体*和[链接](https://www.cc98.org)

- 列表项目
- 另一个项目

> 引用内容
```

遇到权限、网络或内容显示问题时，查看[常见问题](/faq/troubleshooting)。
