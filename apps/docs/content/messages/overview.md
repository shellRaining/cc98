# 查看通知和私信

## 查看未读消息

登录后，顶部消息图标会显示未读总数。点击图标可以进入消息中心，未读数量按以下分类统计：

- **回复我的**：其他用户回复了你的主题或帖子。
- **@ 我的**：其他用户在内容中提及了你。
- **系统通知**：论坛发送的系统消息。
- **我的私信**：你和其他用户之间的私信。

打开对应分类后，页面会更新已读状态。多个浏览器标签页同时打开时，未读数量可能短暂不同步，刷新或等待实时更新即可。

## 使用私信

进入“我的私信”后，可以查看已有会话。选择用户后，右侧显示双方的消息记录，并可继续发送新消息。

发送失败时，输入内容会保留。确认网络和登录状态后再重试，不要连续发送相同内容。

## 调整消息设置

消息中心的“消息设置”用于控制通知偏好。保存后，设置会写入账号，在其他设备登录时继续生效。

如果消息数量没有更新、列表加载失败或私信无法发送，可以查看[常见问题](/faq/troubleshooting)。个人内容和关注入口见[管理个人内容](/account/personal-content)。
